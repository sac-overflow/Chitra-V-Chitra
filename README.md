# Chitra Vichitra Events

> **Run the site from:** `C:\Users\sachh\Desktop\CVC\chitra-vichitra-events`  
> The `#` in the `#CVC` folder name breaks Vite on Windows.

## Easiest way to run

1. Double-click **`START-DEV.bat`** in this folder (opens the runnable copy), **or**
2. Open `Desktop\CVC\chitra-vichitra-events` in your editor and run `npm run dev`

After edits here, copy changes to the runnable folder, or work directly in `CVC\chitra-vichitra-events`.

## Sync changes to runnable copy

```powershell
robocopy "." "C:\Users\sachh\Desktop\CVC\chitra-vichitra-events" /E /XD node_modules dist .git
```
